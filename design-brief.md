# マチコエ デザイン依頼

## アプリ概要

「マチコエ」は富士河口湖町の議会議題を住民に届け、意見表明を促すWebアプリです。
ターゲットは**地元住民（30〜70代）**。難しそう・お役所っぽいと思われると敬遠されるため、
**親しみやすく、まず見てみたくなるデザイン**が必要です。

---

## 現状の構成（4画面）

| 画面 | 内容 |
|---|---|
| トップ（議題一覧） | 議題カードを並べる。カードにはキーワードバッジ・タイトル・AI要約・「自分への影響を見る」ボタン |
| 自分ごと化画面 | 「住民として／事業者として」を選ぶと、自分への影響をAIが説明 |
| 意見表明画面 | 賛成／反対／条件付き賛成を選ぶとAIが意見文の下書きを生成。Xへのシェアやパブコメへのコピーができる |
| みんなの声（ダッシュボード） | 賛否の横棒グラフと匿名意見一覧 |

現状のカラーはインディゴ（`#4f46e5`）ベースのモノトーン。全体に無機質・お役所感がある。

---

## デザイン方針

**キーワード：** 開かれた民主主義 / 地域への愛着 / 気軽に参加できる / 清潔感

### 色の方向性

- メインカラーは**富士山・河口湖をイメージした青緑系**（例：ティール `#0d9488` や空色 `#0ea5e9`）を検討
- サブカラーに**温かみのある朱色・橙色**（例：`#f97316`）を差し色として使いたい
- 背景は白〜ごく薄い青白。暗くしない
- グレー系テキストは読みやすさ優先

### カードデザイン

- キーワードバッジに色をしっかりつけて視覚的に楽しく
- カードにごくわずかな影か、左側に色のアクセントラインを入れたい
- ホバー時にふわっと浮く感じ

### ヒーローエリア（トップページ上部）

- 今は `h1` タイトルだけでさびしい
- キャッチコピー「街の声を、議会へ。」を大きく見せる短いヒーローセクションがほしい
- 富士山や河口湖をイメージしたイラストか、淡いグラデーション背景で温かみを出したい

### 意見表明画面のスタンスボタン

- 賛成＝緑系、反対＝赤系、条件付き賛成＝黄色系は維持
- 今は地味。選択時にもっと気持ちよくハイライトされてほしい（塗りつぶし＋少しの影）

### フッター

- 「行政を攻撃するツールではない」という注記は大事なので残す
- ただし圧迫感なく、小さくおさめる

### フォント

- 日本語は読みやすさ重視
- ロゴ「マチコエ」部分だけ少し個性的な太字でもOK

## 実装前提

- Next.js 14 + Tailwind CSS v4
- コンポーネントはシンプルに（shadcn/ui などは未使用）
- モバイルファーストで、スマホでも崩れない
